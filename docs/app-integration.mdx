## Quick start
1. Make sure you have [created a Sigyl API key](/docs/API-Keys)
2. Go to your desired MCP server page in the [registry](https://www.sigyl.dev/marketplace)
3. Click **Connect** in the MCP server page
4. Configure your secrets for the server
5. Select the app you want to install the MCP to
6. Run the provided command in your terminal to install the server
7. Restart your app and **start using the MCP!**
