Sigyl requires you to have an API key in order to connect to any of the MCP servers on our registry and to use our SDK.

## Getting API keys
1. Log in to your Sigyl account using GitHub
2. Navigate to the **Dashboard**
3. Go to the **Secrets** tab
4. Under the **API Keys** tab, create your key and save it
